Study Efficiency Analyzer
(i)Overview- I built this project to understand how effective my study time actually is. I often found myself studying for long hours but not knowing if it was productive. So I created a simple tool to track both time and performance.

(ii) Features-
Task tracking
 Pomodoro + normal timer
 Efficiency calculation
 Graph visualization

 (iii)How to Run
	1.	Open the project folder
	2.	Run:python main.py
	

(iv) Future Scope
•	Better graphs
	•	Study suggestions
	•	GUI version


(v) Problem statement
Most study tools only track time or tasks, but they don’t show whether that time was useful or not.

(vi)Solution
This project lets users log study sessions with time spent and a self-evaluated score. It then calculates efficiency (score divided by time) to give a basic idea of productivity.
